Hyle AWS Cognito 회원가입 테스트

📌 현재 상황

로컬 테스트 앱에서는 라우팅이 작동하지 않아 회원가입 페이지로 직접 이동할 수 없습니다.

🚀 해결 방법

1. 메인 앱 실행 (라우터 포함)
flutter run -d chrome

⚠️ 주의: -t lib/main_test_local.dart 없이 실행하세요!

2. 앱이 열리면 URL 확인

• 로그인 화면이 바로 나타날 수 있습니다

• 안 나타나면 URL 끝에 #/signup 추가

3. 회원가입 진행

• 실제 이메일 사용

• 비밀번호: Test1234!

• 이메일로 받은 인증 코드 입력

💡 대안

AWS Console에서 직접 테스트 계정 생성:

  1. AWS Console → Cognito → User pools
  2. "Create user" 버튼 클릭
  3. 이메일과 임시 비밀번호 설정
  4. 생성된 계정으로 로그인 테스트
